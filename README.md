# Sales-data-analysis
Sales data analysis using python and numpy

A Python script that generates and analyzes 5 years of 
monthly sales data using NumPy.

## What it does
- Generates random sales data across 5 years and 12 months
- Calculates total sales per year
- Calculates monthly averages across all years
- Identifies the year with highest Q4 sales
- Identifies the best performing month overall

## Skills demonstrated
- NumPy array operations
- Axis-based aggregations (sum, mean)
- Python loops and zip()
- Real-world data analysis logic

pip install numpy
